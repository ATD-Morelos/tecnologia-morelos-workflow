# MANUAL DE USO — REFRENDO ANUAL VEHICULAR PERSONA FÍSICA 2026

## Introducción

Con el compromiso de brindarte certeza y facilitar tu trámite ante Movilidad,

- Qué revisar antes de comenzar.

## Antes de iniciar

### Cuenta Llave MX (obligatorio)

Para realizar el Refrendo Digital es indispensable contar con una cuenta activa
de Llave MX. Si aún no tienes una, deberás crearla antes de iniciar.
Para registrarte necesitarás: CURP, número de teléfono y correo electrónico.

Verifica lo siguiente:

- [ ] Ser el propietario registrado del vehículo.
- [ ] No contar con adeudos de años anteriores.
- [ ] Tener póliza de seguro vigente.
- [ ] Contar con INE vigente.

### Documentos y datos a la mano

- Datos de Llave MX (correo y teléfono) para iniciar sesión.
- Número de placas.
- Últimos 5 dígitos del número de serie.
- INE del propietario (anverso y reverso).
- Comprobante de pago (solo si realizaste el pago antes de iniciar el trámite digital).

> **Importante:** Si ya iniciaste un trámite anteriormente, ingresa a "Mis Trámites"
> y retómalo. No generes uno nuevo.

## Paso a Paso

{% steps %}

{% step title="Ingresa al portal" %}
Ve a [digital.morelos.gob.mx/tramitesdigitales/refrendo-anual](https://digital.morelos.gob.mx/tramitesdigitales/refrendo-anual).
{% /step %}

{% step title="Inicia sesión con Llave MX" %}
Usa tu correo y contraseña de Llave MX. Si aún no tienes cuenta, créala en
[llave.gob.mx](https://llave.gob.mx) con tu CURP, teléfono y correo electrónico.
{% /step %}

{% step title="Captura los datos de tu vehículo" %}
Ingresa tu número de placa y los últimos 5 dígitos del número de serie exactamente
como aparecen en tu tarjeta de circulación, sin espacios ni guiones.
{% /step %}

{% step title="Verifica tus datos y póliza de seguro" %}
El sistema validará adeudos y datos del vehículo. Revisa que la información mostrada
sea correcta. 👉 Si el sistema no valida automáticamente tu póliza de seguro,
ingrésala manualmente en el apartado correspondiente y espera hasta 48 horas;
recibirás un correo de validación.
{% /step %}

{% step title="Sube tu INE" %}
Fotografía el anverso y reverso de tu INE. La plataforma confirmará tu identidad
como propietario. Asegúrate de que la imagen sea clara, sin sombras ni recortes.
{% /step %}

{% step title="Elige tu método de pago" %}
Selecciona: línea de captura, tarjeta bancaria o pago en ventanilla.
Si ya realizaste el pago, selecciona "Ya he realizado mi pago" e ingresa
tu línea de captura. Considera que puede tardar hasta 72 horas en reflejarse.
{% /step %}

{% step title="Espera la validación y descarga tu tarjeta" %}
Una vez confirmado el pago, el sistema procesará tu trámite automáticamente.
Descarga tu Tarjeta de Circulación Digital desde "Mis Trámites" > "Completados".
{% /step %}

{% /steps %}

> **Consideraciones:**
> - Si ya existe un trámite iniciado, deberás concluirlo. Generar uno nuevo puede provocar inconsistencias.
> - Si no aparecen los datos de tu vehículo o propietario, comunícate con la Coordinación de Movilidad y Transporte.
> - Si eres persona moral, consulta el [manual para persona moral](https://tecnologia.morelos.gob.mx/refrendo-moral) para el proceso correspondiente.

## Preguntas Frecuentes

{% accordion title="¿Qué documentos necesito para hacer mi refrendo en línea?" %}
Necesitarás: tarjeta de circulación actual, póliza de seguro vigente, identificación
oficial vigente, número de placa y número de serie del vehículo.
{% /accordion %}

{% accordion title="¿Hay multa si no refrendo a tiempo?" %}
Circular sin refrendo puede generarte una multa. Te recomendamos renovarlo antes
del 31 de marzo de 2026 desde [digital.morelos.gob.mx/refrendo](https://digital.morelos.gob.mx/refrendo).
{% /accordion %}

{% accordion title="¿Qué hago si el sistema no valida mi póliza de seguro?" %}
Si ves el mensaje "No se puede continuar sin Póliza Vigente", el sistema no identificó
tu póliza. Cárgala manualmente: ingresa el nombre de tu aseguradora y número de póliza,
y espera la validación por correo (máximo 42 horas). Una vez aprobada, regresa a tu
trámite en proceso en [digital.morelos.gob.mx/refrendo](https://digital.morelos.gob.mx/refrendo).
{% /accordion %}

{% accordion title="¿Ya pagué pero aún no se refleja. ¿Qué hago?" %}
Verifica que tu banco haya confirmado el pago y espera hasta 72 horas para la
actualización automática. No generes una nueva línea de captura. Si después del
plazo no se actualiza, comparte tu comprobante de pago para revisión.
{% /accordion %}

{% accordion title="¿Qué hago si no puedo descargar mi Tarjeta de Circulación Digital?" %}
No realices un segundo pago. Ingresa a "Mis Trámites" > "Completados" e intenta
descargar nuevamente. Si no aparece disponible, comparte tu comprobante para revisión.
{% /accordion %}

{% accordion title="¿Me aparece que el trámite ya existe pero yo no lo concluí?" %}
Esto indica que ya hay un trámite iniciado con tus placas en el ejercicio actual.
Ingresa a "Mis Trámites" y retoma únicamente el primer trámite generado. No es
necesario iniciar otro.
{% /accordion %}

{% accordion title="¿Cuánto tiempo tarda en completarse el trámite?" %}
Si no hay validaciones pendientes, puede completarse de inmediato. Considera que
el pago puede tardar hasta 72 horas en reflejarse.
{% /accordion %}

{% accordion title="¿Qué hago si el portal se queda cargando o no responde?" %}
Cierra sesión e ingresa nuevamente. Intenta desde otro navegador (Chrome o Edge)
o dispositivo, y verifica tu conexión a internet.
{% /accordion %}
