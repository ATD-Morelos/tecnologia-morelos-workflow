# Changelog

Todos los cambios notables de este plugin se documentan en este archivo.

El formato sigue [Keep a Changelog](https://keepachangelog.com/es-ES/1.1.0/) y este proyecto adhiere a [Semantic Versioning](https://semver.org/lang/es/).

## [0.1.2] - 2026-04-21

### Added
- Nueva skill `voice` con `SKILL.md`, `checklist.md`, `ejemplos.md`, `patrones.md`, `principios.md` y 4 templates (`comunicado.md`, `cta-corto.md`, `notificacion.md`, `respuesta-ciudadana.md`).

### Changed
- `skills/kb-publish/SKILL.md` — refinamientos en el flujo de publicación.
- `skills/review/review-checklist.md` — ítems adicionales de revisión.
- `.mcp.json` y `CLAUDE.md` — actualizaciones menores.
- Versión ahora declarada únicamente en `.claude-plugin/marketplace.json` (fuente de verdad para plugins con `source` relativo, según best practices de Anthropic).
